How to use the data

Converting JSON to DataFrame using jsonlite package in R

# Load the library; ensure that you have installed it first
# install.packages('jsonlite', dependencies=TRUE, repos='http://cran.rstudio.com/')
library(jsonlite)
# Give the input file name to the function. Place the JSON file in the working directory
data <- fromJSON("data.json")

Converting JSON to DataFrame using pandas library in Python 

# Import the pandas library
import pandas as pd
# Give the input file name to the function.
